HTML (Hyper Text Markup Language) is a computer-based markup language used to present content on the Internet. The language is standardized by the W3 consortium. All HTML documents must start with a DOCTYPE declaration.

        <!DOCTYPE html>
        <html>
        <head>
        <title>Title of the document</title>
        </head>
        
        <body>
        The content of the document......
        </body>
        
        </html>

In the browser we will see such a note.

The content of the document......


Elements and attributes.The basic unit of HTML is the element. Like XML, an HTML element has a name and can have any number of attributes. An item can contain text and other items. Both the text and the daughter elements can usually be repeated and followed in any order.
